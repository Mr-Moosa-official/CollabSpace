# **App Name**: CollabSpace

## Core Features:

- Real-time Document Co-editing: Enable multiple users to edit documents simultaneously with synchronized updates and cursor tracking.
- Instant Project Chat: Provide a dedicated chat interface within each project for real-time communication and file sharing.
- Live Activity Updates: Display a stream of live updates showing recent activities by project participants, such as edits, comments, and task assignments.
- User Authentication and Authorization: Secure access to the workspace with user accounts, role-based permissions, and project-level access controls.
- Version History and Document Recovery: Automatically save version history, allowing users to revert to previous document versions.
- AI-Powered Content Suggestions: Suggest relevant content snippets or sources using an AI tool as users type.
- Project and Course Pages: Dedicated pages for projects and courses with integrated document access and collaboration tools.

## Style Guidelines:

- Primary color: Muted blue (#6495ED) for a calming professional atmosphere.
- Background color: Soft gray (#F0F0F0) provides a neutral backdrop that emphasizes content.
- Accent color: Light blue (#ADD8E6) is used for highlights and interactive elements, providing subtle visual cues.
- Body font: 'Inter' sans-serif for a modern, clean and neutral readability.
- Headline font: 'Space Grotesk' sans-serif headlines to provide a computerized, techy, scientific feel. Used for emphasis, short amounts of text.
- Use minimalistic line icons for interface elements, promoting clarity and a modern aesthetic.
- Implement smooth transitions and subtle animations for user interactions, improving user experience.
